# es-html-parser

HTML parser is a HTML parser for static analitics perpose

## Install

```
yarn add es-html-parser
```

or

```
npm install es-html-parser
```

## Usage

```js
import { parse } from "es-html-parser";

const root = parse("");
```

## Development

**Clone repository**

```
$ git clone
```

**Install dependencies**

```
$ yarn
```

**Run tests**

```
$ yarn test
```

**Run lint**

```
$ yarn lint
```

**Format codes**

```
$ yarn format
```
